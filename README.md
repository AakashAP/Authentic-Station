# Authentic Station

Authentic Station is a beautifully crafted restaurant website design created to showcase your restaurant, its menu, and provide an excellent online presence for your customers. This project is built with HTML5, CSS3, and Bootstrap, making it responsive and visually appealing.

This is a Introductory Web Project with Bootstrap.

## Features
- **Homepage** \
  A captivating introductory view with exciting offers, using flexible layouts and cards.\
  For Introductory View and Excitimg Offers. Using Flex and Cards.
- **Menu** \
  Display your restaurant's menu with style, using flexible layouts and cards.\
  For Previewing the Menu. Using Flex and Cards.
- **Gallery** \
  Showcase your restaurant's best content through a well-organized gallery.\
  For Displaying the best content. Using Flexboard Content Alignment.
- **Miscellaneous Info** \
  A section for miscellaneous information, using flexible board content justification.\
  For Miscellaneous Information. Using Flexboard Content Justification.
- **About** \
  Provide contact information and an easy-to-use contact form for your customers.\
  For Contact Information and Contact Form.


## Technology
**HTML5, CSS3, BOOTSTRAP**
