# Authentic Station

Authentic Station is a website design for Restuarant website. The design includes Homepage, Menu, Gallery, About Us -

**Homepage** - For Introductory View and Excitimg Offers. Using Flex and  Cards.\
**Menu** - For Previewing the Menu. Using Flex and Cards.\
**Gallery** - For Displaying the best content. Using Flexboard content alignment.\
**Mis. Info** - For Miscellaneous Information. Using Flexboard content justification.\
**About** - For Contact Information and Contact Form.

This is a Introductory Web Project with Bootstrap.
